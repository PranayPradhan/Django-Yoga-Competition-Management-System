Role-Based Navigation Guide

1) Registration User
Dashboard -> Participants -> Register Participants
Dashboard -> Participants -> Edit Participants
Dashboard -> Participants -> List Registered Participants

Dashboard -> School -> Register School
Dashboard -> School -> Edit School
Dashboard -> School -> List Registered Schools

2) Score Entry User
Dashboard -> Score Data Entry -> Yoga Sub-Category
Dashboard -> Score Data Entry -> Champion of Champions

3) Report User
Dashboard -> Reports -> Event MC Call Sheets
Dashboard -> Reports -> Champion of Champions MC Call Sheets
Dashboard -> Reports -> Competition Results

4) Administrator (Admin Only)
Dashboard -> Master Tables -> Master Tables - CRU
    -> Age Category
    -> Gender
    -> Yoga Sub-Category
    -> Standard
    -> Section
    -> Food Preference

Dashboard -> Settings -> Set Settings
